# Composing handlers

Coming soon. Two related concepts in one chapter: **named handlers** with `handler N : Effect { ... }` and `with N` for reuse, and **chained handlers** with `} handle E1 { ... } handle E2 { ... }` for blocks that perform multiple independent effects. Includes a worked example combining a logging effect with a counting effect to demonstrate composition.
